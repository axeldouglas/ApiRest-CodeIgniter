# ApiRest-CodeIgniter
Desenvolvimento de uma aplicação de Tasks utilizando CodeIgniter como RestServer.

## Requerimentos

1. PHP 5.4 ou superior
2. CodeIgniter 3.0+

_Nota: configurações utilizando PHP 7 apresentam erros_
